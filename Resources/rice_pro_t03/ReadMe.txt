Sample Model
Rice Grassfield - PRO

============================================================

　This model can be used to learn about applied examples of Extension Interpolation and Inverted Masks.

　This model faces sideways as in a fighting game.
　Extension Interpolation is used to rotate a Warp Deformer of the summoning circle.
　Inverted Masks are used to see through a forelock that covers the eye and express the flickering of a flame.
